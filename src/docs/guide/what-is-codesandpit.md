


# 什么是codesandpit？


您可以编写HTML&CSS、js的示例,就像普通的js插件一样，引入到您的项目或者文档中，即可用于演示现场的demo



## 浏览器兼容

不支持IE浏览器,具体可以在源码文件中查阅[.browserslistrc](https://github.com/ajiho/codesandpit/blob/main/.browserslistrc)
